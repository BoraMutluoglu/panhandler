Panhandler is a tool for sharing PAN-OS configurations and configuration sets called Skillets. These configs are stored and shared via Git repositories. In addition to PAN-OS, Panhandler can also manage other types of Skillets, such as Terraform templates, REST APIs, or generic network device configuration templates.

To find out more about Skillets or discover new Skillets, see the live community page here.

To learn more about building Configuration and Validation Skillets, see the Skillet Builder Documentation.
